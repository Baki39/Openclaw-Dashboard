# UltraClaw Dashboard

Professional Android App Development Dashboard - God-Mode 2026

## Features

- 📊 **Overview** - Total apps, revenue, downloads, active apps
- 📱 **Apps History** - All deployed apps with status and metrics
- 📈 **Analytics** - Detailed per-app analytics with charts
- 📥 **Review Queue** - Apps pending approval
- 🔄 **Update Center** - Request updates for existing apps

## Tech Stack

- React + Vite
- Tailwind CSS
- Recharts
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Push to main branch to deploy to GitHub Pages automatically.
